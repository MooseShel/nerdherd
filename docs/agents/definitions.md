# Specialist Agent Definitions

## 1. Social Matchmaking Agent
- **Track**: Phase 1, Item 1
- **Focus**: AI-driven user compatibility, vector embeddings, and icebreakers.
- **Skills**: Vector search, LLM personality matching, profile analysis.

## 2. Smart Spot Agent
- **Track**: Phase 1, Item 2
- **Focus**: Real-time spot summaries, aggregate location data, and summarized reviews.
- **Skills**: Data aggregation, text summarization, Flutter UI (Map/Cards).

## 3. Academic Assistant Agent
- **Track**: Phase 2
- **Focus**: Smart Scheduling and "NerdBot" Tutor Helper.
- **Skills**: Calendar integration, conversational AI, conflict resolution logic.

## 4. Engagement Agent
- **Track**: Phase 3
- **Focus**: Wrapped stories, gamification, and global feeds.
- **Skills**: Data visualization, social feed logic, gamification mechanics.

## 5. Infrastructure Agent
- **Track**: Foundation
- **Focus**: Supabase setup, Vector database migrations, and Glassmorphism 2.0 design system.
- **Skills**: Database architecture, Flutter CustomPainters, advanced CSS/Flutter themes.
